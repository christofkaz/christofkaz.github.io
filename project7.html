Introduction

I created this web app in Python using modules such as Flask, SQLalchemy, WTForms.

Additionaly all the user data including the likes for quotes data is stored in the PostgreSQL format. The data for quotes is sourced from a file.

User authentication

Upon registration, your password becomes encrypted with the help of the Werkzeug module, which hashes the password before storing it in the database, making it secure during a data breach

People who simply want to use the API, don't need to be authenticated.

Functions for authenticated users

Logged in users gain the ability to give likes to quotes which affects the "Best Rated Quotes" section

API Functionality

The most important part of the project is its API functionality

To access nearly 50 000 quotes, users can view the API documentation for help

The API response is a neatly formatted array with json objects. Each object in the response contains the quote, its author and the category of that quote.
