Introduction

I created this web app in Python using modules such as Flask, SQLalchemy, WTForms. Additionaly all the user and tasks data is stored in PostgreSQL format.

The frontend side of this webapp was partly constructed using templates provided by Bootstrap.

Most of the functionality is covered by Flask but JavaScript functions are also a part of this project.

User authentication

Upon registration, your password becomes encrypted with the help of the Werkzeug module, which hashes the password before storing it in the database, making it secure during a data breach

Functions for authenticated users

Logged in users gain an access to their own to-do list, which stores all the to-do list data in a PostgreSQL database

The to-do list allows their owners to change the order of the tasks, mark tasks as completed, delete tasks, or rename the task list.
