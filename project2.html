Introduction

I developed this quiz game web app using mainly JustPy. That way, I was able to build both the frontend and backend of the app with just Python code.

The purpose behind this project was to hone my knowledge with Python's OOP and its more advanced functions

All the quiz data in the project is being requested via an API

How to play

The user can choose the difficulty and select one of the 12 available quizes with questions randomized each time the quiz is opened.

The player answers questions and has his score displayed at the bottom of the page. Only first correct answer grants him a point

Final score gets displayed after the quiz ends and the quiz can be partaken again
